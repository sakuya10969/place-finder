# デザイン方針 — place-finder

## コンセプト

「近くのおすすめスポットをジャンルごとに、楽しく・直感的に探せる」体験を目指す。
白背景をベースに、ジャンルごとの固有カラーとアイコンで画面に彩りを持たせつつ、
メインカラー（ブルー）で全体の統一感を保つ。

キーワード: カラフル・直感的・ジャンル識別・地図連動

## カラーパレット

### ベースカラー

| 用途 | 色 | HEX | 説明 |
|---|---|---|---|
| 背景 | 白 | `#FFFFFF` | ページ全体、カード背景 |
| メインカラー | ブルー | `#228BE6` | ヘッダー、主要ボタン、アクティブ状態 |
| サブカラー | ダークブルー | `#1864AB` | セクション見出し、ナビゲーション |
| テキスト | チャコール | `#1A1A1A` | 本文テキスト |
| ボーダー | ライトグレー | `#E5E7EB` | カード枠線、セパレーター |
| サーフェス | ウォームグレー | `#F8F9FA` | セクション背景、ホバー状態の背景 |

### ジャンル別アクセントカラー

各ジャンルに固有の色とアイコンを割り当て、バッジ・カード・地図マーカーで一貫して使用する。

| ジャンル | カラー名 | HEX | アイコン（@tabler/icons-react） |
|---|---|---|---|
| レストラン | コーラルオレンジ | `#E8590C` | `IconToolsKitchen2` |
| カフェ | アンバーブラウン | `#D6A01E` | `IconCoffee` |
| コンビニ | エメラルドグリーン | `#0CA678` | `IconBuildingStore` |
| 薬局 | スカイブルー | `#1C7ED6` | `IconFirstAidKit` |
| その他 / すべて | ブルー | `#228BE6` | `IconMapPin` |

ジャンルカラーは以下の箇所で統一的に使用する:
- ジャンルフィルタのバッジ背景・枠線
- 店舗カードの左ボーダーまたはアクセントライン
- 地図上のマーカー色
- 店舗カード内のジャンルアイコン

### ステータスカラー

| 状態 | HEX | 用途 |
|---|---|---|
| 営業中 | `#2B8A3E` | 営業中バッジ、ドットインジケーター |
| 営業時間不明 | `#868E96` | 不明時のバッジ |
| エラー | `#C92A2A` | バリデーションエラー、通知 |
| 成功 | `#2B8A3E` | 成功通知 |

### スコア表現カラー

おすすめスコアの高低を色で直感的に伝える。

| スコア帯 | カラー | HEX |
|---|---|---|
| 高スコア（0.7 以上） | グリーン | `#2B8A3E` |
| 中スコア（0.4〜0.7） | アンバー | `#E8590C` |
| 低スコア（0.4 未満） | グレー | `#868E96` |

## アイコン運用方針（@tabler/icons-react）

プロジェクト全体で `@tabler/icons-react` を統一的に使用する。

### 主要アイコンマッピング

| 用途 | アイコン | サイズ目安 |
|---|---|---|
| 現在地 | `IconCurrentLocation` | 20px |
| 距離 | `IconMapPin` | 14–16px |
| 評価（星） | `IconStarFilled` | 14–16px |
| 営業中 | `IconClock` | 14px |
| 検索 | `IconSearch` | 18px |
| フィルタ | `IconFilter` | 18px |
| 地図タブ | `IconMap` | 18px |
| リストタブ | `IconList` | 18px |
| エラー | `IconAlertCircle` | 20px |
| リトライ | `IconRefresh` | 18px |
| ナビゲーション | `IconCompass` | 24px（ヘッダーロゴ横） |

### アイコンスタイルルール

- ジャンルアイコンはジャンル固有カラーで着色する
- 汎用アイコン（距離・評価など）はチャコール `#1A1A1A` またはメインカラー `#228BE6` を使用
- アイコンサイズは隣接テキストと視覚的にバランスが取れるサイズにする
- stroke 幅は Tabler デフォルト（1.5）を維持し、統一感を保つ

## タイポグラフィ

- 本文: system-ui, 14〜16px, チャコール
- 見出し: 18〜24px, ダークブルーまたはチャコール, font-weight 700
- ラベル・補足: 12〜13px, チャコール（opacity 0.6〜0.7）
- 行間は 1.5〜1.6 を基準にゆとりを持たせる
- ジャンル名はジャンルカラーで表示してもよい（バッジ外のテキストとして使う場合）

## レイアウト

- Mantine の AppShell を使い、ヘッダー + メインコンテンツの構成
- ヘッダーはダークブルー背景に白テキスト。`IconCompass` + サービス名を配置
- メインコンテンツは白背景の 1 カラム構成
- 最上部のカード内に、左側: 見出し + ジャンルフィルタ + 検索半径、右側: 地図（380×260px 程度の横長長方形）を横並びで配置
- その下に店舗カード一覧を縦に並べる
- 余白は Mantine の spacing トークン（md, lg）を基準に統一

## コンポーネント方針

### ヘッダー
- ダークブルー `#1864AB` 背景、白テキスト
- 左に `IconCompass`（24px, 白）+ サービス名
- 右に `IconCurrentLocation` で現在地取得状態を表示

### ジャンルフィルタ（バッジ群）
- 各ジャンルバッジにジャンル固有カラーとアイコンを表示
- 未選択時: outline バリアント（ジャンルカラーの枠線 + ジャンルカラーのテキスト・アイコン）
- 選択時: filled バリアント（ジャンルカラーの背景 + 白テキスト・アイコン）
- 「すべて」バッジはブルー `#228BE6` を使用

### 店舗カード
- 白背景 + ライトグレー枠線
- カード左端にジャンルカラーの 4px ボーダー（アクセントライン）を表示
- 店舗名の横にジャンルアイコン（ジャンルカラー、16px）
- 距離は `IconMapPin`、評価は `IconStarFilled` を添えて表示
- 営業中バッジはグリーン `#2B8A3E` filled、不明時はグレー outline
- 選択中のカードはメインカラーの枠線 + サーフェスカラー `#F8F9FA` の背景
- スコアバッジはスコア帯に応じた色で表示

### ボタン
- メインアクション: ブルー filled
- セカンダリ: ブルー outline
- アイコン付きボタンを積極的に使い、視認性を高める（例: `IconRefresh` + 「再試行」）

### 地図
- Google Maps JavaScript API で描画
- 検索カード内の右上に 380×260px 程度の横長長方形として配置
- マーカーはジャンルカラーで色分け（ジャンルごとに異なる色のマーカー）
- 選択中のマーカーはサイズを大きくするか、バウンスアニメーションで強調
- 情報ウィンドウにはジャンルアイコン + 店舗名 + 評価を簡潔に表示

### ローディング
- Mantine Skeleton でカード形状のプレースホルダーを表示
- Loader のカラーはブルー

### 通知
- エラー: 赤 `#C92A2A` + `IconAlertCircle`
- 成功: グリーン `#2B8A3E` + `IconCircleCheck`

## テーマ設定

Mantine の createTheme で以下を定義する:

- primaryColor: ブルー系のカスタムカラー（brand パレットを `#228BE6` ベースに再構成）
- ジャンルカラーは Mantine のカスタムカラーとして追加登録し、コンポーネントの `color` prop で参照可能にする
- デフォルト radius: md
- カラースキーム: light 固定
- フォント: system-ui

### 追加カスタムカラー（Mantine テーマに登録）

ジャンルカラーを Mantine のカラーパレット形式（10段階）で登録し、
`color="genre-restaurant"` のように prop で指定できるようにする。

- `genre-restaurant`: コーラルオレンジ系
- `genre-cafe`: アンバーブラウン系
- `genre-convenience`: エメラルドグリーン系
- `genre-pharmacy`: スカイブルー系

## モバイル対応

- Mantine のブレークポイント（sm, md, lg）に従ったレスポンシブ設計
- デスクトップと同じ 1 カラムレイアウトを維持（タブ切り替え不要）
- タッチ操作を考慮し、バッジ・ボタンのタップ領域は最低 44px を確保
- ジャンルフィルタは横スクロール可能な Group で表示（画面幅に収まらない場合）

## やること / やらないこと

やること:
- ジャンルごとの固有カラー + アイコンで視覚的な識別性と楽しさを提供する
- メインカラー（ブルー）で全体の統一感を維持する
- `@tabler/icons-react` を全面的に活用し、テキストだけに頼らない UI にする
- 営業状態・スコアを色で直感的に伝える
- Mantine コンポーネントをベースに、カスタム CSS は最小限にする
- 一貫した余白・角丸・フォントサイズを維持する

やらないこと:
- ダークモード対応（初期段階）
- 独自のコンポーネントライブラリを構築する
- グラデーション、ドロップシャドウの多用、過剰な装飾
- ジャンルカラー同士を混ぜたり、1つの要素に複数のジャンルカラーを適用する
