# 機能一覧 — place-finder

ユーザーが何をできるかを定義する。技術的な実装詳細はここには含めない。

## MVP 機能

### 1. 現在地の取得

ユーザーの現在の GPS 位置情報を取得し、検索の基準点として利用する。

### 2. 店舗検索

現在地の周辺にある店舗を検索し、一覧として表示する。

### 3. ジャンル別フィルタリング

飲食店、カフェ、コンビニなど、ジャンルを指定して検索結果を絞り込む。

### 4. おすすめ順ソート

距離・評価・営業状況などの条件に基づいて、ユーザーにとって有用な順序で店舗を並べ替える。

### 5. 店舗基本情報の閲覧

各店舗について以下の情報を確認できる:
- 店舗名
- ジャンル
- 現在地からの距離
- 評価（レーティング）
- 営業状況（営業中 / 閉店中）

### 6. 地図表示

検索結果の店舗を地図上にマーカーで表示し、位置関係を視覚的に把握できる。

## MVP 対象外の機能（将来候補）

| 機能 | 概要 |
|---|---|
| ユーザーアカウント | 登録・ログイン・プロフィール管理 |
| お気に入り | 気に入った店舗を保存して後から確認 |
| 履歴 | 過去に検索・閲覧した店舗の履歴表示 |
| レビュー投稿 | ユーザーが店舗に対して口コミ・評価を投稿 |
| AI おすすめ | ユーザーの行動履歴・嗜好に基づくパーソナライズ推薦 |
| プッシュ通知 | 近くのおすすめ店舗の通知 |
| 多言語対応 | 日本語以外の言語での利用 |
